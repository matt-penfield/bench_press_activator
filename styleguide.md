# Style Guide — Slalom Brand

Reference for applying Slalom's visual identity to this project.

## Colors

### Primary
- **Slalom Blue (Primary):** `#0C62FB`
- **Dark Navy:** `#001B36`
- **White:** `#FFFFFF`

### Secondary
- **Cyan:** `#00C7B1`
- **Bright Green:** `#A0E82C`
- **Magenta:** `#E91E8C`
- **Orange:** `#FF6B35`
- **Purple:** `#6B3FA0`

### Neutrals
- **Dark Gray:** `#2D2D2D`
- **Medium Gray:** `#6B7280`
- **Light Gray:** `#F4F5F7`
- **Border Gray:** `#E5E7EB`

## Typography

- **Primary Font:** Slalom Sans (fallback: `"Inter", "Helvetica Neue", Arial, sans-serif`)
- **Headings:** Bold (700), clean and concise
- **Body:** Regular (400), 16px base
- **Small/Labels:** Medium (500), 12–14px, uppercase for category labels

## Spacing

- Base unit: `8px`
- Component padding: `16px`, `24px`, `32px`
- Section gaps: `32px–48px`

## Border Radius

- Cards/containers: `8px–12px`
- Badges/pills: `20px`
- Buttons: `4px–8px`

## Shadows

- Subtle: `0 1px 4px rgba(0, 0, 0, 0.08)`
- Elevated: `0 4px 16px rgba(0, 0, 0, 0.12)`
- Modal: `0 8px 32px rgba(0, 0, 0, 0.2)`

## Tone & Voice

- Clear, confident, human
- Action-oriented, not jargon-heavy
- Inclusive and empowering

## UI Principles

- Clean, spacious layouts with generous whitespace
- Bold pops of color used sparingly for emphasis
- Dark navy for headers/nav, white for content areas
- Data-forward: let content lead, minimize chrome
